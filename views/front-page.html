Codemiration

noun
respect and warm approval for well written code
"I have the greatest codemiration for the code in the project"

Comments

Aim for zero comments.

Discussion

  • Only comments that describe why code is doing what it is doing are valid, such as business rules.
  • If you have to state what your code is doing then your code is not clear enough.
  • Over time comments may become irrelevant if not updated along with code changes, doubling up maintenence costs. If not updated they become misleading.
